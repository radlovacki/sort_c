# Sorting algorithms in C language

## Selection sort

Алгоритам сортирања избором (енгл. *Selection Sort*) ради на принципу
поновљеног проналажења минимума (или максимума) у неуређеном делу низа. Низ се
концептуално дели на два дела — **уређени**, који се налази на почетку и у
старту је празан, и **неуређени**, који у старту обухвата цео низ.

У сваком пролазу кроз неуређени део, алгоритам проналази најмањи елемент и
поставља га на прво слободно место у уређеном делу, тј. замењује га са
елементом на позицији одмах иза границе уређеног дела. Након сваког пролаза,
граница уређеног дела се помера за једно место удесно, а неуређени део се
смањује за један елемент.

Поступак се понавља све док неуређени део не остане са само једним елементом,
који је аутоматски на исправној позицији. За низ од $n$ елемената потребно је
тачно $n-1$ пролаза. Кључна особина алгоритма је да у сваком пролазу врши
највише једну замену, без обзира на стање низа.

Задатак: Корисник уноси број елемената низа са стандардног улаза. Програм
генерише задати динамички низ, попуњава га насумично генерисаним целим
бројевима, исписује га на стандардни излаз, сортира га, исписује сортирани
низ на стандардни излаз, на крају ослобађа меморију.
